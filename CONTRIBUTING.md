# Contributing

Gracias por considerar contribuir a este proyecto.

- Reporta bugs abriendo un issue con pasos para reproducir.
- Para mejoras o características, abre un issue y luego un PR.
- Sigue el estilo de código existente y escribe tests cuando sea posible.

Proceso recomendado:
1. Fork del repositorio
2. Crea una rama (`feature/mi-cambio`)
3. Haz commits claros y atómicos
4. Abre un Pull Request contra `main`

Gracias :)
